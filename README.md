# Resume Portfolio Website

A modern and responsive resume portfolio built using HTML, CSS, and JavaScript.

## Features

✔ Responsive Design
✔ Download Resume PDF
✔ Print Resume
✔ Edit Resume
✔ Animated Skill Bars
✔ Smooth Scrolling
✔ Modern UI
✔ Mobile Friendly

## Technologies

- HTML5
- CSS3
- JavaScript
